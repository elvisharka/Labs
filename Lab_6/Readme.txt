How to run the app
    1) How to run the server
        Main class to run the server: com.example.server.core.Server
        You need to set `Program arguments` as `study_groups.csv`
            1. Open Intellij IDEA
            2. Go to Run -> Edit Configurations
            3. In Program arguments text box type `study_groups.csv` (without quotes)
            4. Run main method in Server class
    2) How to run the client
        Main class to run the server: com.example.client.core.Client. Just run it
